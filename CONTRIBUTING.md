# Contributing to Parquet

Thank you for your interest in contributing! Here's how you can help.

## Ways to Contribute

### 🐛 Report Issues
Found a bug? [Open an issue](https://github.com/oii-nasif/Parquet/issues) with:
- Clear description of the problem
- Steps to reproduce
- Browser/screen details (if relevant)

### 💡 Suggest Features
Have an idea? [Open an issue](https://oii-nasif/Parquet/issues) with:
- Feature description
- Use case
- Why it would help learners

### 📝 Improve Content
- Correct incorrect answers or explanations
- Add more questions to existing quizzes
- Improve clarity of explanations
- Fix typos or formatting issues

### 🎨 Improve Design
- Fix responsive layout issues
- Improve accessibility
- Enhance UI/UX

### 🔧 Code Contributions
1. Fork the repository
2. Create a branch for your changes
3. Make your changes
4. Submit a pull request

## Pull Request Guidelines

- Keep changes focused and minimal
- Follow existing code style
- Test your changes before submitting
- Include a clear description of what you changed and why

## Questions?

Feel free to open an issue for any questions!
